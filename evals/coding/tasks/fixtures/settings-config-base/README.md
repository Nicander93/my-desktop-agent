# 设置配置 / Settings Configuration

创建 `config/settings.json`，内容必须与下方 JSON 完全一致（两个空格缩进，末尾保留换行）。

Create `config/settings.json` with exactly this JSON document (two-space indentation and a trailing newline):

```json
{
  "enabled": true,
  "maxRetries": 3,
  "regions": ["ap-southeast-1", "eu-west-1"]
}
```

不要修改 verifier。 / Do not edit the verifier.
