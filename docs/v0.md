# Desktop Agent V0 PRD

## 1. 产品概述

### 产品名称（暂定）

Desktop Agent

### 产品定位

一个运行在本地桌面的 AI 工作台。

用户通过自然语言下达任务，Agent 自动调用本地工具完成代码开发、文档处理、文件整理等工作。

V0 不追求 Computer Use，不控制鼠标键盘，不操作任意 GUI 软件。

重点打造：

* Coding Agent
* 本地文件 Agent
* Office Agent
* MCP Agent

### 产品愿景

让用户通过一个统一的 Chatbox，完成大部分数字化办公和开发任务。

---

# 2. 核心用户

典型任务：

* 修复项目 Bug
* 分析代码

* 编写单元测试
* 生成文档

## 办公人员

典型任务：

* 生成 Word

* 整理文件
* 汇总 Excel

## AI 重度用户

典型任务：

* 调用 MCP
* 管理本地知识库
* 自动执行重复工作

# 3. V0 功能范围

## 包含

### Chat 对话

用户通过自然语言描述任务。

例如：

```text
帮我分析这个 SpringBoot 项目

帮我整理下载目录

根据需求生成汇报 PPT

读取这个 Word 并总结
```

---

### Coding Agent

支持：

* 读取项目
* 搜索代码
* 修改文件
* 生成代码
* 生成测试
* 执行命令
* 查看执行结果

典型流程：

```text
用户
 ↓
Agent
 ↓
读取项目
 ↓
分析代码
 ↓
修改文件
 ↓
执行测试
 ↓
返回结果
```

---

### 文件整理 Agent

支持：

* 文件分类
* 文件重命名
* 文件移动
* 文件清理建议
* 重复文件识别

示例：

```text
整理我的下载目录

按图片、视频、文档分类
```

---

### Word Agent

支持：

* 创建 docx
* 修改 docx
* 提取内容
* 格式化内容

示例：

```text
根据这份需求生成设计文档
```

---

### PPT Agent

支持：

* 创建 pptx
* 自动分页
* 生成大纲
* 插入图片

示例：

```text
生成项目汇报 PPT
```

---

### Excel Agent

支持：

* 创建 xlsx
* 数据统计
* 数据透视分析
* 表格汇总

---

### MCP

支持：

* 本地 MCP Server
* 远程 MCP Server

用户可自行扩展能力。

---

# 4. 不在 V0 范围

## Computer Use

不支持：

* 鼠标点击
* 键盘模拟
* 屏幕识别
* GUI 自动化

---

## 多 Agent

暂不支持：

```text
Planner Agent
Coder Agent
Reviewer Agent
```

统一使用单 Agent Runtime。

---

## 云端同步

不支持：

* 账号系统
* 多设备同步
* 云端存储

全部本地运行。

# 5. 系统架构

```text
Desktop UI
    ↓
Local API
    ↓
Agent Runtime
    ↓
Tool Router
    ├─ File Tool
    ├─ Shell Tool
    ├─ Code Tool
    ├─ Docx Tool
    ├─ Ppt Tool
    ├─ Excel Tool
    └─ MCP Tool
```

---

# 6. Tool 规范

每个 Tool 必须包含：

```ts
interface Tool {
  name: string;

  description: string;

  parameters: JsonSchema;

  execute(input): Promise<ToolResult>;
}
```

统一注册：

```ts
toolRegistry.register(tool);
```

Agent 不直接操作系统。

全部通过 Tool。

---

# 7. Session 机制

每个会话维护：

```ts
Session
 ├─ messages
 ├─ artifacts
 ├─ toolCalls
 ├─ files
 └─ context
```

支持：

* 历史消息
* Tool 调用记录
* 文件引用

---

# 8. Artifact 机制

Agent 生成的内容统一保存为 Artifact。

类型：

```text
Markdown
Code
Docx
Pptx
Xlsx
Image
Json
```

用户可直接打开。

# 10. V0 成功标准

用户能够完成以下任务：

1. 对本地代码项目进行分析和修改
2. 自动整理指定目录
3. 生成 Word 文档
4. 生成 PPT
5. 调用 MCP 工具
6. 查看 Agent 执行过程
7. 查看生成文件和结果

达到以上能力即可发布 V0。
