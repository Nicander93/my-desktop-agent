# Desktop Agent

本地桌面 AI 工作台（Electron + React + Agent SDK）。

## 开发

```bash
pnpm install
pnpm dev
```

## 质量检查

```bash
pnpm check    # typecheck + lint + dep-check + knip + test
pnpm lint
pnpm test
```
